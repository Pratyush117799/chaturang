# Game History Database

Save your completed Chaturanga games here. When you choose "Save game to database" before closing the app (or when auto-download is off and you're prompted), the game moves and winner are downloaded as a file. Place those downloaded files in this folder to build your game history database.

File naming: `chaturanga_YYYY-MM-DD_HH-mm-ss_TeamN.txt` (or `PlayerN` for single mode).
