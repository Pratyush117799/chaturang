# Chaturanga v1.0.3.3 Vision & Roadmap

## 1. Game Data Analyser (The "Seer" Engine)
The next evolution of the benchmark system will include a built-in game logs analyser.
- **Model Prediction**: Given a set of game logs, the analyser will estimate the ELO rating of each player (even anonymous ones) based on move quality, capture efficiency, and tactical blunders.
- **Insight Engine**: It will highlight key "turning points" in a game and provide insights into the logic behind bot moves (e.g., "Bot chose Ratha capture over King safety due to Greedy-ELO-200 constraints").
- **MD/Text Exports**: The analyser will generate detailed Markdown reports summarizing game trends across hundreds of simulations.

## 2. Modern "Object-Based" UI
Moving away from simple static images, the game board will be redesigned to feel like a premium digital board game (similar to chess.com or xiangqi.com).
- **Physical Interaction**: Pieces will no longer be mere image tags; they will be defined as interactive objects that can be clicked, dragged, and animated smoothly across the grid.
- **Visual Excellence**:
  - High-resolution piece sets with authentic Indian aesthetic.
  - Dynamic board highlights for last move, legal targets, and "check" states.
  - Smooth transitions and sound effects for captures and dice rolls.

## 3. Persistent Analytics Vault
A dedicated section in the website to store, search, and review all "Historical Benchmarks" and "Master Games" in a searchable database format.
